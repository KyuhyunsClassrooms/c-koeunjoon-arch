1. 전체 흐름

(도서 관리 프로그램의 전체 작동 로직을 의사 코드로 작성)

시작
반복 (사용자가 '4. 종료'를 선택할 때까지):
    메뉴 출력 (1. 등록, 2. 전체 목록, 3. 검색, 4. 종료)
    사용자 입력 받기

    만약 입력이 1이면:
        책 등록 함수 호출

    만약 입력이 2이면:
        전체 도서 목록 출력 함수 호출

    만약 입력이 3이면:
        검색 함수 호출

    만약 입력이 4이면:
        반복 종료
반복 끝
종료

2. 핵심 기능 1: 책 등록 함수
시작
    만약 현재 등록된 책 수가 최대 수량을 초과하면:
        "등록 불가" 메시지 출력
        함수 종료

    사용자에게 책 제목 입력받기
    사용자에게 저자 입력받기
    사용자에게 가격 입력받기

    구조체 배열에 제목/저자/가격 저장

    현재 책 수 +1 증가

    "등록 완료" 메시지 출력
종료

3. 핵심 기능 2: 제목으로 책 검색 함수
시작
    만약 현재 등록된 책이 0권이면:
        "등록된 책 없음" 출력
        함수 종료

    사용자에게 검색할 제목 입력받기

    found 변수를 0으로 초기화

    전체 책 목록을 0번째부터 마지막까지 반복:
        만약 책 제목에 검색 키워드가 포함되어 있으면:
            책 제목/저자/가격 출력
            found = 1로 변경

    만약 found가 여전히 0이면:
        "일치하는 책 없음" 출력
종료
